Linux Ubuntu Terminal Interface - Это проект на Python который позволяет создавать интерфейс с нужными инструментами для удобного пользования Linux.
В данной версий (0.0.0-alpha) Доступны лишь основные инструменты для работы с файлами в системе: Проводник и редактор файлов - Editor.
LUTI требует установленного python, так как все написано на нем и работает лишь на нем.
Теперь к запуску. Чтобы запустить файл нужно ввести простую команду: python {Путь/к/LUTI.py}, или python3 {Путь/к/LUTI.py} (Если у вас python3).
После появиться окно с выбором нужных вам программ!
